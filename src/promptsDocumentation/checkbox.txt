# Component Specification

## Overview
The Checkbox component is a binary selection control for choosing one or multiple options from a set.

**Key Features:**
- Binary selection (checked/unchecked)
- Support for multiple checkboxes in groups
- Light and dark theme support
- SVG checkmark with smooth animations
- Disabled state handling
- Fully accessible with semantic HTML
- Token-driven styling (NEVER hardcode values)

## NPM Package Information

**Package:** `@atomchat.io/components-astro@2.0.4`
**Component path:** `src/atoms/Checkbox.astro`
**Dependencies:**
- `@atomchat.io/tokens@1.0.3` (design tokens)
- `@atomchat.io/css@0.2.3` (compiled CSS from tokens)

**Optional dependencies (for animations):**
- `@atomchat.io/animations` (GSAP animation functions)
- `gsap` (animation library)

## Props API

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| name | string | — | Input name attribute |
| value | string | — | Input value attribute |
| checked | boolean? | false | Controlled checked state |
| defaultChecked | boolean? | false | Default checked state (uncontrolled) |
| disabled | boolean? | false | Disables checkbox interaction |
| theme | "light" \| "dark" | "light" | Visual theme variant |
| class | string? | — | Additional CSS classes |
| onChange | function? | — | Change event handler |

### Children
| Slot | Description |
|------|-------------|
| default | Checkbox label text (required) |

## Animation System (if enabled)

### Data Attributes Contract
- `data-checkbox` — Root element
- `data-checkbox-custom` — Custom indicator
- `data-checkbox-check` — SVG checkmark

### GSAP Animation Implementation

**Check animation (SVG path drawing):**
```typescript
// On check
gsap.to('[data-checkbox-check]', {
  opacity: 1,
  scale: 1,
  duration: 0.3,
  ease: 'back.out(1.7)'
})

// On uncheck
gsap.to('[data-checkbox-check]', {
  opacity: 0,
  scale: 0.5,
  duration: 0.2,
  ease: 'power2.in'
})
```

### Motion Preferences
```css
@media (prefers-reduced-motion: reduce) {
  .checkbox__custom,
  .checkbox__check {
    transition-duration: 0.01ms !important;
  }
}
```

## Accessibility Requirements

**Semantic HTML:**
- Use `<label>` wrapping all elements
- Use native `<input type="checkbox">`
- Custom indicator marked with `aria-hidden="true"`
- Label text visible and associated with input

**Keyboard navigation:**
- Tab to focus checkbox
- Space to toggle checked state
- Focus-visible styling required
