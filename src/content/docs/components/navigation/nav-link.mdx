## Overview

NavLink is a minimal navigation anchor styled for horizontal navbars and menus. It uses a bottom border to indicate hover and active states, bold weight for active, and a focus ring for keyboard navigation.

**Key Features:**
- 5 states: inactive · hover · active · focused · disabled
- Bottom border on hover and active
- Bold weight on active
- `data-active="true"` or `.nav-link--active` for active state
- `aria-disabled` or `.nav-link--disabled` for disabled
- Respects `prefers-reduced-motion`

## Anatomy

```html
<a class="nav-link" href="/docs">Documentation</a>

<!-- Active -->
<a class="nav-link nav-link--active" href="/components" data-active="true">
  Components
</a>

<!-- Disabled -->
<a class="nav-link nav-link--disabled" aria-disabled="true">
  Coming Soon
</a>
```

## API Reference

### States

| Class / Attribute | State | Visual |
|-------------------|-------|--------|
| *(none)* | Default | No border |
| `:hover` | Hover | Bottom border `--stroke-xs` |
| `.nav-link--active` / `data-active="true"` | Active | Bottom border + bold |
| `:focus-visible` | Focused | Blue border-radius outline |
| `.nav-link--disabled` / `aria-disabled="true"` | Disabled | `opacity-20`, no pointer |

## Usage

### Horizontal Nav Bar

```html
<nav aria-label="Main navigation" style="display: flex; gap: var(--m); align-items: center;">
  <a class="nav-link" href="/">Home</a>
  <a class="nav-link nav-link--active" href="/components" data-active="true"
    aria-current="page">Components</a>
  <a class="nav-link" href="/foundations">Foundations</a>
  <a class="nav-link nav-link--disabled" aria-disabled="true">Changelog</a>
</nav>
```

### With JS Active State

```javascript
const links = document.querySelectorAll('.nav-link');
const current = window.location.pathname;

links.forEach(link => {
  const isActive = link.getAttribute('href') === current;
  link.classList.toggle('nav-link--active', isActive);
  link.dataset.active = isActive;
  if (isActive) link.setAttribute('aria-current', 'page');
});
```

## Accessibility

- Use `aria-current="page"` on the active link
- Disabled links: `aria-disabled="true"` + `tabindex="-1"` to remove from tab order
- Wrap in `<nav aria-label="...">` for landmark recognition

```html
<nav aria-label="Primary navigation">
  <a class="nav-link" href="/" aria-current="page">Home</a>
  <a class="nav-link" href="/docs">Docs</a>
  <a class="nav-link nav-link--disabled" aria-disabled="true" tabindex="-1">
    API (soon)
  </a>
</nav>
```
