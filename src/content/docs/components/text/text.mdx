## Overview

The Text component renders flexible typography elements for body text, captions, labels, and auxiliary content. It supports single-line truncation, multi-line clamping, text balancing, and optional GSAP animations.

### Key Features

- **Flexible Element Types**: Render as `<p>`, `<span>`, `<div>`, or `<li>` for semantic variety
- **5 Size Options**: Body, caption, label, label-small, and footnote sizes
- **3 Weight Options**: Regular, medium, and bold font weights
- **Truncation Support**: Single-line with ellipsis or multi-line clamping
- **Text Balancing**: Optional text-wrap: balance for improved readability
- **6 GSAP Animations**: Masked text, fade, scramble, and highlight effects
- **3 Alignment Options**: Left, center, right text alignment
- **Custom Colors**: Support for CSS color values or inherit
- **Lightweight**: Simple, flexible text wrapper component

## Installation

To use the Text component in your project, install it from the Atom Design System package:

```bash
npm install @atomchat.io/components-astro
# or
pnpm add @atomchat.io/components-astro
```

Then import and include the CSS tokens and GSAP library (for animations):

```astro
---
import { Text } from '@atomchat.io/components-astro'
import '@atomchat.io/components-astro/dist/tokens.css'

// Optional: For animations
import gsap from 'gsap'
import SplitText from 'gsap/SplitText'
gsap.registerPlugin(SplitText)
---
```

## Anatomy

The Text component renders a flexible element with typography modifiers:

```html
<p class="text text--body text--regular text--left" data-text>
  Body text content
</p>

<!-- With truncation -->
<p class="text text--body text--truncate" data-text style="overflow: hidden; text-overflow: ellipsis; white-space: nowrap;">
  Truncated text...
</p>

<!-- With animation -->
<span class="text text--body text--regular" data-text data-text-animate="fade-up-load" data-text-trigger="load" data-text-split="lines">
  Animated text
</span>
```

**DOM Structure:**
- `p`, `span`, `div`, or `li` - Flexible element (determined by `as` prop)
- `.text` - Root component class
- `.text--{size}` - Size modifier (body, caption, label, label-small, footnote)
- `.text--{weight}` - Font weight modifier (regular, medium, bold)
- `.text--{align}` - Text alignment modifier (left, center, right)
- `.text--truncate` - Applied when truncate=true (single-line ellipsis)
- `.text--clamp` - Applied when truncate is a number (multi-line clamp)
- `.text--balance` - Applied when balance prop is true (text-wrap: balance)
- `data-text` - Component identifier
- `data-text-animate` - Animation type (if animate prop provided)
- `data-text-trigger` - Animation trigger mode (scroll, load)
- `data-text-split` - SplitText split type (always "lines")
- `data-text-delay` - Animation delay in seconds
- `style="--text-clamp-lines: N"` - Custom property for line clamping
- `style="color: [value]"` - Applied for custom colors

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `"p" \| "span" \| "div" \| "li"` | `"p"` | HTML element type |
| `size` | `"body" \| "caption" \| "label" \| "label-small" \| "footnote"` | `"body"` | Text size for different contexts |
| `weight` | `"regular" \| "medium" \| "bold"` | `"regular"` | Font weight emphasis |
| `color` | `string` | `"inherit"` | Text color (CSS value or inherit) |
| `align` | `"left" \| "center" \| "right"` | `"left"` | Text alignment |
| `truncate` | `boolean \| number` | `false` | Truncation mode (true = single-line, number = line clamp) |
| `balance` | `boolean` | `false` | Enable text-wrap: balance for readability |
| `animate` | `"masked-lines" \| "masked-words" \| "fade-up-load" \| "scramble-load" \| "scramble-scroll" \| "highlight-scroll" \| false` | `false` | GSAP animation type |
| `trigger` | `"scroll" \| "load"` | `"scroll"` | Animation trigger mode |
| `delay` | `number` | `0` | Animation delay in seconds |
| `class` | `string` | `""` | Additional CSS classes |

### Slots

| Slot | Description |
|------|-------------|
| `default` | Text content (required) |

### Data Attributes

| Attribute | Values | Purpose |
|-----------|--------|---------|
| `data-text` | Present (no value) | Component identifier |
| `data-text-animate` | Animation type or undefined | Animation selector for GSAP |
| `data-text-trigger` | "scroll", "load", or undefined | Animation trigger mode |
| `data-text-split` | "lines" | SplitText split type |
| `data-text-delay` | Delay in seconds or undefined | Animation delay value |

## Accessibility

### Semantic HTML

- **Use correct elements**: `<p>` for paragraphs, `<span>` for inline, `<li>` for list items
- **Respect semantics**: Don't use `<div>` for body text when `<p>` is appropriate
- **Screen reader friendly**: Text content is always readable

### Truncation Accessibility

- **Truncated content**: Users should be able to access full content through other means (hover, expand, link to full content)
- **Don't hide critical information**: Ensure essential content is not truncated

### Animation Accessibility

- **Respects prefers-reduced-motion**: Animations should be disabled for users with motion sensitivities
- **Non-blocking**: Animations should not prevent content from being read immediately

### Best Practices

```html
<!-- Good: Semantic paragraph -->
<Text as="p">This is a paragraph of content</Text>

<!-- Good: Inline with span -->
<Text as="span">Inline text within a sentence</Text>

<!-- Good: Truncate with accessible fallback -->
<Text truncate={2} title="Full text content here">
  Text that might be truncated...
</Text>

<!-- Avoid: Hiding critical info with truncation -->
<Text truncate>Important information that should always be visible</Text>

<!-- Avoid: Using div for body text unnecessarily -->
<Text as="div">This should probably be a paragraph</Text>
```
