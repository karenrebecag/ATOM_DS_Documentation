## Overview

The TextArea component is a multi-line text input with built-in custom scrollbar, supportive text, character counter, and the same state system as TextField (filled, error, disabled). Available in two sizes: m and xl.

**Key Features:**
- 2 sizes (m, xl)
- Custom scrollbar (hidden native scrollbar)
- Supportive text with icon
- Character counter
- States: default · hover · focus · filled · error · disabled
- `resize: none` — height controlled via tokens
- Fully token-driven

## Anatomy

```
┌───────────────────────────────┐
│ .textarea                     │
│  .textarea__label             │  ← Label
│  .textarea__field             │  ← Container (border, fixed height)
│  │  .textarea__input          │  ← Native <textarea> (no resize)
│  │  .textarea__scrollbar      │  ← Custom scrollbar track
│  │    .textarea__scrollbar-thumb │
│  .textarea__bottom            │
│  │  .textarea__supportive     │  ← Helper text
│  │  .textarea__counter        │  ← Character counter
└───────────────────────────────┘
```

**DOM Structure:**
```html
<div class="textarea textarea--m">
  <label class="textarea__label">Message</label>
  <div class="textarea__field">
    <textarea class="textarea__input" placeholder="Write your message..."></textarea>
    <div class="textarea__scrollbar">
      <div class="textarea__scrollbar-thumb"></div>
    </div>
  </div>
  <div class="textarea__bottom">
    <span class="textarea__supportive">
      <span class="textarea__supportive-text">Describe your issue in detail</span>
    </span>
    <span class="textarea__counter">0 / 500</span>
  </div>
</div>
```

## API Reference

### Modifier Classes

| Class | Description |
|-------|-------------|
| `textarea--m` | Medium size |
| `textarea--xl` | Extra large size |
| `textarea--filled` | Applied when textarea has content |
| `textarea--error` | Error state |
| `textarea--disabled` | Disabled state |

### States

| State | Trigger | Visual |
|-------|---------|--------|
| **Default** | Initial | Gray border |
| **Hover** | `:hover` | Darker border |
| **Focus** | `:focus` | Primary border + shadow |
| **Filled** | `.textarea--filled` | Filled border color |
| **Error** | `.textarea--error` | Red border |
| **Disabled** | `.textarea--disabled` | Muted bg, no-drop cursor |

## Usage

### Basic TextArea

```html
<div class="textarea textarea--m">
  <label class="textarea__label">Bio</label>
  <div class="textarea__field">
    <textarea class="textarea__input" placeholder="Tell us about yourself..."></textarea>
    <div class="textarea__scrollbar">
      <div class="textarea__scrollbar-thumb"></div>
    </div>
  </div>
</div>
```

### With Counter

```html
<div class="textarea textarea--m">
  <label class="textarea__label">Message</label>
  <div class="textarea__field">
    <textarea class="textarea__input" maxlength="500"></textarea>
    <div class="textarea__scrollbar">
      <div class="textarea__scrollbar-thumb"></div>
    </div>
  </div>
  <div class="textarea__bottom">
    <span class="textarea__supportive">
      <span class="textarea__supportive-text">Be concise and clear</span>
    </span>
    <span class="textarea__counter" id="char-counter">0 / 500</span>
  </div>
</div>
```

### Error State

```html
<div class="textarea textarea--m textarea--error textarea--filled">
  <label class="textarea__label">Feedback</label>
  <div class="textarea__field">
    <textarea class="textarea__input">x</textarea>
    <div class="textarea__scrollbar">
      <div class="textarea__scrollbar-thumb"></div>
    </div>
  </div>
  <div class="textarea__bottom">
    <span class="textarea__supportive">
      <span class="textarea__supportive-text">Please provide at least 20 characters</span>
    </span>
  </div>
</div>
```

  </span>
</div>
