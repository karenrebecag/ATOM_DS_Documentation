## Overview

The LinkButton component renders a semantic `<a>` element styled as a text link with interactive states. Unlike the Button component, it has a single visual style focused on inline navigation — no variants, just two sizes and clean hover/focus/loading states. The right icon nudges on hover to reinforce directionality.

**Key Features:**
- 2 sizes: `s` (12px) and `l` (16px)
- Single visual style — no variants
- Disabled and loading states
- Icon support (left and/or right)
- Right icon nudge on hover
- Shimmer animation on loading text
- Fully token-driven styling
- Accessible with `aria-disabled`, `aria-busy`, and `tabIndex` management

## Anatomy

```
┌────────────────────────────────┐
│  .link-button (<a>)            │  ← Base element
│  ┌──────────────────────────┐  │
│  │ .link-button__icon       │  │  ← Optional left icon
│  │ .link-button__label      │  │  ← Label text
│  │ .link-button__icon       │  │  ← Optional right icon (nudges on hover)
│  └──────────────────────────┘  │
└────────────────────────────────┘
```

**DOM structure (default):**
```html
<a class="link-button link-button--l" href="/about" data-link-button>
  <span class="link-button__label">
    Learn more
  </span>
</a>
```

**DOM structure (with icons):**
```html
<a class="link-button link-button--l" href="/about" data-link-button>
  <span class="link-button__label">
    <span class="link-button__icon"><!-- icon-left --></span>
    Learn more
    <span class="link-button__icon"><!-- icon-right --></span>
  </span>
</a>
```

**DOM structure (loading):**
```html
<a class="link-button link-button--l link-button--loading link-button--disabled"
   aria-disabled="true" aria-busy="true" tabindex="-1" data-link-button>
  <span class="link-button__loading-content">
    <span class="link-button__loading-text" data-shimmer>Loading</span>
  </span>
</a>
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | — | Navigation URL |
| `size` | `"s" \| "l"` | `"l"` | Text size — `s` is 12px, `l` is 16px |
| `disabled` | `boolean` | `false` | Disables interaction, applies `aria-disabled` and `tabIndex=-1` |
| `loading` | `boolean` | `false` | Shows shimmer loading text, disables interaction |
| `external` | `boolean` | `false` | Adds `target="_blank" rel="noopener noreferrer"` (Astro only) |
| `iconLeft` | `ReactNode` | — | Icon before the label (React/Vue/Angular) |
| `iconRight` | `ReactNode` | — | Icon after the label — nudges on hover (React/Vue/Angular) |

**Astro slots:**

| Slot | Description |
|------|-------------|
| `default` | Label text |
| `icon-left` | Icon before the label |
| `icon-right` | Icon after the label — nudges on hover |

## Sizes

| Size | Font size | Use case |
|------|-----------|----------|
| `s` | 12px (`--link-button-font-size-s`) | Dense layouts, captions, secondary actions |
| `l` | 16px (`--link-button-font-size-l`) | Default, body text, primary actions |

---

## States

| State | Class | Behavior |
|-------|-------|----------|
| Default | `.link-button` | Standard interactive link |
| Hover | `:hover` | Background changes, right icon nudges `translateX` |
| Focus | `:focus-visible` | Border + shadow ring |
| Pressed | `:active` | Background changes |
| Disabled | `.link-button--disabled` | `cursor: not-allowed`, `aria-disabled`, `tabIndex=-1` |
| Loading | `.link-button--loading` | Shimmer text, icons hidden, `aria-busy` |

## Accessibility

- `aria-disabled="true"` + `tabIndex="-1"` when disabled — keeps element in DOM but out of tab order
- `aria-busy="true"` during loading state
- Right icon is decorative — no ARIA needed
- Use descriptive link text, avoid "click here" or "read more" without context

```html
<!-- Good: descriptive -->
<a class="link-button" href="/pricing">View pricing</a>

<!-- Good: external with context -->
<a class="link-button" href="https://github.com"
   target="_blank" rel="noopener noreferrer"
   aria-label="View on GitHub (opens in new tab)">
  GitHub
</a>
```
