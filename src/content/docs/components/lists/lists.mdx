## Overview

The Lists system provides two styled list-item components — `bullet-item` and `number-item` — that replace bare `<li>` elements with token-driven typography and consistent spacing. Both support inline `<strong>` emphasis and `<a>` links.

**Key Features:**
- `.bullet-item` — custom `•` bullet via `::before`
- `.number-item` — ordered list item (counter via CSS or HTML `<ol>`)
- `<strong>` inside: medium weight, primary color
- `<a>` inside: underline with hover transition
- `margin-bottom: var(--s)` between items, zero on `:last-child`
- Fully token-driven typography

## Anatomy

```html
<!-- Unordered list -->
<ul style="list-style: none; padding: 0;">
  <li class="bullet-item">First item</li>
  <li class="bullet-item">Second item with <strong>emphasis</strong></li>
  <li class="bullet-item">Item with <a href="#">a link</a></li>
</ul>

<!-- Ordered list (number-item) -->
<ol style="list-style: none; padding: 0; counter-reset: list-counter;">
  <li class="number-item">First step</li>
  <li class="number-item">Second step</li>
</ol>
```

## API Reference

### BulletItem

| Selector | Description |
|----------|-------------|
| `.bullet-item` | List item with `•` before pseudo-element |
| `.bullet-item strong` | Medium weight, `--fg-primary` color |
| `.bullet-item a` | Underline + hover color transition |
| `.bullet-item:last-child` | Zero bottom margin |

## Design Tokens

```css
/* BulletItem */
--font-body              /* Font family */
--text-body              /* Font size */
--font-weight-regular    /* Default weight */
--font-weight-medium     /* <strong> weight */
--font-weight-bold       /* Bullet character weight */
--line-height-body
--fg-secondary           /* Item text color */
--fg-primary             /* Bullet + <strong> + link color */
--duration-quarter       /* Link hover transition */
```

## Accessibility

- Always use semantic `<ul>` or `<ol>` as the parent element
- `list-style: none` on the parent is acceptable — screen readers still announce list role
- If using `list-style: none` with CSS-driven bullets, add `role="list"` on the `<ul>` for VoiceOver compatibility

```html
<ul role="list" style="list-style: none; padding: 0;">
  <li class="bullet-item">Item one</li>
  <li class="bullet-item">Item two</li>
</ul>
```
