## What is AtomChat DS?

AtomChat Design System is a comprehensive monorepo built with modern web standards. It provides independently publishable npm packages that work together to deliver tokens, styles, and animations for building consistent and scalable interfaces.

<CardGrid stagger>
  <Card title="@atomchat.io/tokens" icon="setting">

  </Card>
  <Card title="CDN Ready" icon="rocket">

  <Card title="GSAP Animations" icon="star">

## Three-Layer Token Architecture

AtomChat DS uses a hierarchical token system:

1. **Primitive Layer** — Raw foundation values (`--primitive-blue-500`)
2. **Semantic Layer** — Intent-driven tokens (`--bg-primary`, `--text-body-m`)
3. **Component Layer** — Component-specific tokens (`--buttons-bg-primary-enabled`)

## Quick Start

### Via CDN (Fastest)

```html
<!-- Add to your HTML <head> -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@atomchat.io/tokens@1.0.0/build/css/tokens.css">

<!-- Use tokens in your CSS -->
<style>
  .my-component {
    background: var(--bg-primary);
    color: var(--text-primary);
    padding: var(--gap-m);
  }
</style>
```

### Via npm

```bash
pnpm add @atomchat.io/tokens
```

```css
@import '@atomchat.io/tokens/build/css/tokens.css';
```

### Theme Switching

```html
<html data-theme="light">  <!-- or "dark" -->
```
